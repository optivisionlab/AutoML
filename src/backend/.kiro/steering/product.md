# Tổng Quan Sản Phẩm HAutoML

HAutoML (Hyper-processors Automated Machine Learning) là nền tảng AutoML mã nguồn mở được phát triển bởi sinh viên Trường Công Nghệ Thông Tin và Truyền Thông, Đại Học Công Nghiệp Hà Nội.

## Mục Đích

Cung cấp giải pháp No Code & Low Code giúp người dùng không chuyên về công nghệ có thể xây dựng mô hình học máy mà không cần kiến thức kỹ thuật sâu.

## Tính Năng Chính

- Tự động huấn luyện mô hình học máy và tối ưu hóa siêu tham số
- Hỗ trợ nhiều thuật toán tìm kiếm: Grid Search, Genetic Algorithm, Bayesian Search
- Kiến trúc huấn luyện phân tán sử dụng Kafka và các worker node
- Giao diện REST API và Gradio UI
- Quản lý phiên bản và kích hoạt mô hình
- Quản lý bộ dữ liệu với MinIO storage
- Xác thực người dùng (local và Google OAuth)

## Đối Tượng Người Dùng

- Người dùng không chuyên về kỹ thuật cần khả năng ML
- Nhà khoa học dữ liệu tìm kiếm tối ưu hóa mô hình tự động
- Nhà nghiên cứu thử nghiệm các chiến lược tìm kiếm khác nhau

## Điểm Khác Biệt

- Hỗ trợ tiếng Việt đầy đủ
- Lựa chọn thuật toán tìm kiếm linh hoạt
- Xử lý phân tán để mở rộng quy mô
- Tích hợp với kho dữ liệu UCI ML Repository
